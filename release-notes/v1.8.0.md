# Kaspa Sovereign Architect Engine v1.8.0 - Deterministic Sync, Behavioral Evaluation, Transaction Safety, And Live Source Intelligence

This is the prepared repository release for `v1.8.0`. It has not been
published. The latest published GitHub release remains `v1.6.2`, published on
May 26, 2026.

## Source Of Truth

- Treats `skills/public/kaspa-sovereign-architect-engine/` as canonical.
- Adds dry-run-first two-way synchronization with SHA-256 drift reports,
  ignored-file rules, last-sync conflict detection, branch guards, backups,
  deletion propagation, and explicit forced resolution.
- Confirms the previously installed local skill was an unchanged `v1.5.0`
  package, so no older local content was imported over repository improvements.

## Prompt And Knowledge Architecture

- Reduces `SKILL.md` to a 223-line operating kernel.
- Preserves UTXO, DAG, fee, signing, custody, wallet, address-prefix, source,
  network, and completion rules.
- Adds a task-oriented knowledge map and keeps changing Toccata release,
  activation, API, policy, migration, miner, and research detail in references.
- Adds `TRAINING_SOURCES.md` as the protocol-source ladder for official docs,
  Rusty Kaspa, KIPs, builder/R&D channels, and community tooling references.
- Adds `SYSTEM_ARCHITECTURE.md` and `AGENT_TRACE.md` for Plan-Act-Verify
  memory, observability, self-correction, and Web4 autonomy boundaries.
- Adds `docs/toccata.md` as the official/repo-backed Toccata builder guide for
  node, wallet, pool/miner, indexer, covenant, and ZK/lane-proof work.
- Adds `docs/kaspa/toccata-mastery-track.md` and a deterministic
  `scripts/toccata-protocol-drill.mjs --check` runner for post-activation
  mastery across protocol, covenant, indexer, wallet, ZK, sequencing,
  operations, and application lanes.
- Requires protocol, transaction, covenant, sequencing, wallet, indexer, and
  architecture work to cite the governing source before proposing changes.
- Prevents cross-platform adapters from embedding dated protocol facts while
  retaining the shared safety contract.

## Live Source Intelligence

- Adds `scripts/kaspa-source-intelligence.mjs`, a live primary-source snapshotter
  for Rusty Kaspa releases/refs, KIP metadata, docs/research fingerprints, and
  mainnet/TN10/TN12 blockDAG endpoints.
- Writes machine-readable and reviewer snapshots under
  `research-snapshots/source-intelligence/`.
- Adds a skill reference that forces current/latest claims to cite
  `checkedAt`, `factsHash`, release tags, refs, and network identity evidence.
- Treats endpoint failures as unknown evidence, not proof that a feature,
  network, or deployment is absent.

## Behavioral Evaluations

- Adds 16 deterministic cases covering network confusion, scheduled versus
  active behavior, post-threshold activation claims, KIP lifecycle, DAG indexers, UTXO conservation,
  transaction-plan safety gates, custody/signing, Kasware/Kaspium, mass
  conflicts, relay policy, covenant reorgs, endpoint failures, live source
  intelligence, future features, and dated source citation.
- Checks both required concepts and prohibited claims.
- Supports grading bundled references or external captured responses.
- Adds 8 protocol mastery drill cases covering active-mainnet claim boundaries,
  transaction field migration, fee-policy layering, covenant lineage, wallet
  signing previews, sequencing lanes, ZK proof readiness, and protocol answer
  shape.

## Transaction Plan Safety

- Adds `references/transaction-plan-safety.md` for payment-builder and signing
  preview reviews.
- Adds `scripts/lint-transaction-plan.mjs` to reject secret-like fields,
  broadcast-by-default, network/address mismatches, conflicting fee aliases,
  missing mainnet acknowledgements, and invalid value accounting.
- Adds tests for safe unsigned testnet plans and unsafe mainnet/secret/broadcast
  plans.

## Release Discipline

- Adds machine-readable published-versus-repository release metadata.
- Adds CI validation for manifest, README, adapters, release notes, and package
  version consistency.
- Runs synchronization and behavioral tests from both the repository and an
  extracted release artifact.
